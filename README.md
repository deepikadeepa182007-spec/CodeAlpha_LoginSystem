# CodeAlpha Login & Registration System (C++)

## Description
This project implements a simple Login and Registration System using C++.
User credentials are stored in a file and verified during login.

## Features
- User registration
- User login
- File handling for data storage
- Menu-driven program

## Concepts Used
- C++ File Handling
- Functions
- Strings
- Conditional Statements

## How to Run
1. Compile the program
2. Run the executable
3. Choose register or login from menu
